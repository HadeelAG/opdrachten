Reflectie: 
Deze project was wel een beetje moeilijker dan ik dacht HTML vind ik makkelijk maar CSS is wel mijn zwakte punt.
Wat er ging goed was dat ik snel de HTML kon schrijven en de structuur ervan goed te maken.
wat ik denk niet zo goed ging was de CSS ik moest heel veel elementen en selectors en properties opzoeken, 
waarvan ik wist niet zoveel hoe ik moet er mee omgaan heel veel dingen gingen op de foute plek of iets werkte op een aparte manier.
Hoe ik er mee omging is gewoon meer over lezen en uitleg krijgen (bijv. AI, Documention) en natuurlijk zelf proberen.
Natuurlijk is het mischien niet de meest profesioneel website ook en het is niet zo erg geadvanceerd maar voor helemaal zelf doen vind ik het wel prima.
Voor de volgende keer zou ik waarschijnlijk niks anders doen want het ging best vel oke en het was maar proberen en oefenen.
